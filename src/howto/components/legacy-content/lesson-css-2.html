Figma 属性面板 就是 CSS 属性清单

本页讲什么属性面板 = CSS

适合谁设计师 · 0 代码也能懂
目标看到字段就会写属性
01 / 核心

先记住这一句
属性面板 = CSS 清单

核心

选中一个 Figma 图层，右侧那块属性面板，就是它“长什么样”的全部信息。CSS 把同一块信息写成了文本：一行一个属性，冒号后面跟值。字段名几乎一模一样。

02 / 对照

一个按钮，两种描述
Figma ↔ CSS

方式

同一个按钮，左边是 Figma 属性面板，右边是浏览器看到的 CSS。鼠标移到任意一行，两边会同时高亮，它们是同一个东西。

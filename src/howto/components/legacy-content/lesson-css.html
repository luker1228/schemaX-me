先别急着碰布局系统

核心

这一章先不讲布局，只先建立一个基础认识：CSS 主要是在控制页面看起来怎么样。字大一点还是小一点，颜色深一点还是浅一点，卡片是挤还是松，按钮有没有反馈，这些都归 CSS 管。

02 / 练习

直接动手改四种样式
字号、留白、方向、反馈

方式

下面几段都不需要先记语法名词，你只要改一条 CSS，就能立刻看到视觉变化。先建立“属性值一改，界面就变”的手感。

这是控制字号。

font-size 控制文字大小。把 .card-title 从 16px 改到 55px，标题会立刻变得更醒目，层级也会更强。

CSS 
body {
  font-family: system-ui, sans-serif;
  margin: 24px;
  background: #f5f4ee;
  color: #1a1a1a;
}
.card {
  margin: 0;
  padding: 16px;
  border: 1.5px solid #1a1a1a;
  border-radius: 12px;
  background: #fff;
  max-width: 360px;
}
.card-meta { margin: 0 0 12px; color: #6b6b6b; }
.card-body { margin: 0; line-height: 1.6; }
Preview

这是控制留白。

padding 控制内容和边框之间的内边距。把 .card 从 8px 往上调，卡片内部会变得更松，呼吸感也会更强。

CSS 
body {
  font-family: system-ui, sans-serif;
  margin: 24px;
  background: #f5f4ee;
  color: #1a1a1a;
}
.card-title { font-size: 22px; margin: 0 0 8px; }
.card-meta { margin: 0 0 12px; color: #6b6b6b; }
.card-body { margin: 0; line-height: 1.6; }
Preview

这是控制方向。

flex-direction 控制 flex 子元素的排列方向。把 .card-actions 从 row 改成 column，按钮就会从横排切成竖排。

CSS 
body {
  font-family: system-ui, sans-serif;
  margin: 24px;
  background: #f5f4ee;
  color: #1a1a1a;
}
.card {
  padding: 16px;
  border: 1.5px solid #1a1a1a;
  border-radius: 12px;
  background: #fff;
  max-width: 360px;
}
.card-title { font-size: 22px; margin: 0 0 8px; }
.card-meta { margin: 0 0 12px; color: #6b6b6b; }
.card-body { margin: 0 0 16px; line-height: 1.6; }
.card-button {
  padding: 8px 14px;
  border: 1.5px solid #1a1a1a;
  border-radius: 8px;
  background: #fff;
  cursor: pointer;
}
Preview

这是控制反馈。

:hover 表示鼠标移上去时的状态。给 .card-button:hover 改背景或文字颜色，按钮就会出现反馈，页面也会显得更“活”。

CSS 
body {
  font-family: system-ui, sans-serif;
  margin: 24px;
  background: #f5f4ee;
  color: #1a1a1a;
}
.card {
  padding: 16px;
  border: 1.5px solid #1a1a1a;
  border-radius: 12px;
  background: #fff;
  max-width: 360px;
}
.card-title { font-size: 22px; margin: 0 0 8px; }
.card-meta { margin: 0 0 12px; color: #6b6b6b; }
.card-body { margin: 0 0 16px; line-height: 1.6; }
.card-actions { display: flex; gap: 10px; }
.card-button {
  padding: 8px 14px;
  border: 1.5px solid #1a1a1a;
  border-radius: 8px;
  background: #fff;
  cursor: pointer;
}
Preview

这是控制颜色。

color 和 background 控制组件的主色调。这里改的是 .card-button，可以直接试这 5 个色值：#1a1a1a、#4b5563、#0f766e、#b45309、#b91c1c，看按钮气质怎么变化。

CSS 
body {
  font-family: system-ui, sans-serif;
  margin: 24px;
  background: #f5f4ee;
  color: #1a1a1a;
}
.card {
  padding: 16px;
  border: 1.5px solid #1a1a1a;
  border-radius: 12px;
  background: #fff;
  max-width: 360px;
}
.card-title { font-size: 22px; margin: 0 0 8px; }
.card-meta { margin: 0 0 12px; color: #6b6b6b; }
.card-body { margin: 0 0 16px; line-height: 1.6; }
.card-actions { display: flex; gap: 10px; }
Preview

这是分割线。

这里要区分一下：Markdown 里的 --- 或 HTML 里的 <hr>，是你显式写出来的分割线；但这页很多章节之间那条贯穿全局的线，其实是 CSS 根据相邻区块的关系统一画出来的。

CSS 
body {
  font-family: system-ui, sans-serif;
  margin: 24px;
  background: #f5f4ee;
  color: #1a1a1a;
}
.demo-divider-block {
  background: #fff;
  border-radius: 12px;
  padding: 0;
}
.demo-divider-item {
  padding: 18px 0;
}
.demo-divider-line {
  margin: 0 0 18px;
}
Preview

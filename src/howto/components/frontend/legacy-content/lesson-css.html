CSS-01 核心

CSS究竟在控制什么

不需要学习CSS怎么写， 只需要明白CSS究竟可以控制什么

这是控制字号。

font-size 控制文字大小。把 .card-title 从 16px 改到 55px，标题会立刻变得更醒目，层级也会更强。

CSS-03 练习
CSS 
body {
  font-family: system-ui, sans-serif;
  margin: 24px;
  background: var(--paper);
  color: #1a1a1a;
}
.card {
  margin: 0;
  padding: 16px;
  border: 1.5px solid #1a1a1a;
  border-radius: 12px;
  background: #fff;
  max-width: 360px;
}
.card-meta { margin: 0 0 12px; color: var(--text-soft); }
.card-body { margin: 0; line-height: 1.6; }
Preview

这是控制留白。

padding 控制内容和边框之间的内边距。把 .card 从 8px 往上调，卡片内部会变得更松，呼吸感也会更强。

CSS-04 练习
CSS 
body {
  font-family: system-ui, sans-serif;
  margin: 24px;
  background: var(--paper);
  color: #1a1a1a;
}
.card-title { font-size: 22px; margin: 0 0 8px; }
.card-meta { margin: 0 0 12px; color: var(--text-soft); }
.card-body { margin: 0; line-height: 1.6; }
Preview

这是控制方向。

flex-direction 控制 flex 子元素的排列方向。把 .card-actions 从 row 改成 column，按钮就会从横排切成竖排。

CSS-05 练习
CSS 
body {
  font-family: system-ui, sans-serif;
  margin: 24px;
  background: var(--paper);
  color: #1a1a1a;
}
.card {
  padding: 16px;
  border: 1.5px solid #1a1a1a;
  border-radius: 12px;
  background: #fff;
  max-width: 360px;
}
.card-title { font-size: 22px; margin: 0 0 8px; }
.card-meta { margin: 0 0 12px; color: var(--text-soft); }
.card-body { margin: 0 0 16px; line-height: 1.6; }
.card-button {
  padding: 8px 14px;
  border: 1.5px solid #1a1a1a;
  border-radius: 8px;
  background: #fff;
  cursor: pointer;
}
Preview

这是控制反馈。

:hover 表示鼠标移上去时的状态。给 .card-button:hover 改背景或文字颜色，按钮就会出现反馈，页面也会显得更"活"。

CSS-06 练习
CSS 
body {
  font-family: system-ui, sans-serif;
  margin: 24px;
  background: var(--paper);
  color: #1a1a1a;
}
.card {
  padding: 16px;
  border: 1.5px solid #1a1a1a;
  border-radius: 12px;
  background: #fff;
  max-width: 360px;
}
.card-title { font-size: 22px; margin: 0 0 8px; }
.card-meta { margin: 0 0 12px; color: var(--text-soft); }
.card-body { margin: 0 0 16px; line-height: 1.6; }
.card-actions { display: flex; gap: 10px; }
.card-button {
  padding: 8px 14px;
  border: 1.5px solid #1a1a1a;
  border-radius: 8px;
  background: #fff;
  cursor: pointer;
}
Preview

这是控制颜色。

color 和 background 控制组件的主色调。这里改的是 .card-button，可以直接试这 5 个色值：#1a1a1a、var(--text-soft)、var(--svg-green)、var(--amber-ink)、var(--amber-ink)，看按钮气质怎么变化。

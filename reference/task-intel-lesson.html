01 核心 ● 已解锁

HTML 和 Markdown
本质上是同一类东西

HTML 就像 JSON 或 Markdown。JSON 是给程序看的结构化数据，HTML 是给浏览器看的结构化文档。Markdown 用 # 表示标题，HTML 用 <h1>；Markdown 用 - 表示列表，HTML 用 <ul><li>。

概念完全一样，只是写法不同。你不需要记住每个标签怎么写，遇到不会的查一下或者问 AI 就行。

// 战术提示

把 HTML 想成"更啰嗦但更精确的 Markdown"。每次遇到新标签，对应到你已知的 Markdown 语法，找到它就行了。

02 标题 ● 已解锁

这是标题。

Markdown 用 # 数量表示标题层级，HTML 用 <h1> 到 <h6> 标签。一个页面通常只有一个 h1，其他用 h2、h3 组织层级。

MARKDOWN 信号 → HTML 翻译
# 一级标题 ## 二级标题 ### 三级标题
<h1>一级标题</h1> <h2>二级标题</h2> <h3>三级标题</h3>
03 段落 ● 已解锁

这是段落。

Markdown 里空行分隔段落，HTML 需要用 <p> 标签显式包裹每个段落。这是 HTML 比 Markdown "啰嗦"的地方，但也更明确。

MARKDOWN 信号 → HTML 翻译
这是第一段。 这是第二段。 这是第三段。
<p>这是第一段。</p> <p>这是第二段。</p> <p>这是第三段。</p>
04 粗体 ● 已解锁

这是粗体。

Markdown 用 **文字** 加粗，HTML 用 <strong> 标签。两者语义相同：表示重要内容。

MARKDOWN 信号 → HTML 翻译
这是**重要内容**。
这是<strong>重要内容</strong>。

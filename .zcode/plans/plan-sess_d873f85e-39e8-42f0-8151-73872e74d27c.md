# 首页改造成作品集式个人介绍页

## 总体策略
- **保留 `.dossier` 根 class**(它是 `body:has(.dossier)` 浅色主题的触发器,去掉会退回深色主题),但**删除所有 dossier 装饰元素**(topbar、atmosphere、corner brackets、scan line、terminal 窗口、双层硬阴影)。
- 复用现有色彩 token(`--dossier-panel` / `--text` / `--text-soft` / `--yellow` / `--dossier-border` / `--font-display` / `--font-mono`),不发明新色系。
- 作品集卡片**新写**(scoped 在首页),不复用 `DossierAssetCard`(它是 dossier 视觉体系的一部分)。
- 滚动入场动画**从零写**,用 `IntersectionObserver + CSS keyframes`,每个 section 用**不同的戏剧化动效**,且兼容 `prefers-reduced-motion`。
- 所有新增样式写在 `src/pages/index.astro` 的 `<style>` block 里(scoped),不污染 `global.css`;`global.css` 里的 dossier 样式保留不动(howto 可能复用)。

## 技术实现要点

### IntersectionObserver 脚本(首页 `<script>`)
```
- 选取所有 [data-reveal] 元素
- rootMargin: "0px 0px -15% 0px", threshold: 0.15
- 进入视口 → 加 .is-visible,once: true(unobserve)
- prefers-reduced-motion: reduce → 直接全部加 .is-visible 跳过动画
- 数值 count-up:能力矩阵 section 进视口时,逐个把数字从 0 数到目标值
```

### stagger 约定
内部元素挂 `style="--i: 0/1/2..."`,`transition-delay: calc(var(--i) * 80ms)`。

### 入场动效清单(5 种,各不相同)
| Section | 动效 |
|---|---|
| Hero | `clip-path: inset(0 100% 0 0)` → 揭开,逐行 |
| 能力矩阵 | 条形 `width` 生长 + 数值 count-up |
| 技术栈 | `scale + rotate` 翻牌 |
| 最新文章 | 右侧 `translateX` 滑入 |
| 作品入口 | 底部 `translateY + clip-path` 幕布升起 |
